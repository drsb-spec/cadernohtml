Tags Elementares em HTML

Cabeçalhos (títulos)

O cabeçalho é criado com as tags <h1> até <h6>.

Explicação:

  • <h1> define o título principal.
  • <h6> define o título menos importante.

Exemplo:

Título nível 1

Título nível 2

Título nível 3

Parágrafos

O parágrafo em HTML é criado com a tag <p>.

Explicação:

  • <p> inicia o parágrafo.
  • </p> encerra o parágrafo.

Exemplo:

Este é um parágrafo em HTML.

Quebras de linha

A quebra de linha é criada com a tag <br />.

Explicação:

  • Insere uma nova linha dentro do texto, sem criar um novo parágrafo.

Exemplo:

Linha 1
Linha 2
Linha 3

Linhas horizontais

A linha horizontal é criada com a tag <hr />.

Explicação:

  • Usada para separar seções de conteúdo.

Exemplo:

Texto acima da linha

Texto abaixo da linha

Comentários

O comentário é criado com <!-- ... -->.

Explicação:

  • Usado para escrever anotações que não aparecem na página.

Exemplo:

<!-- Isto é um comentário e não será exibido -->
