iframes

A tag <iframe> … </iframe> é usada para criar frames embutidas, ou seja, mostrar uma página dentro de outra.

Explicação:

  • Permite mostrar conteúdo externo dentro da página atual.
  • Exemplo: vídeos, mapas, outros sites embutidos.

Exemplo:

<iframe src="pagina.html" width="600" height="400"></iframe>

Hiperligações em molduras (frames)

Para abrir um link numa moldura específica:

Passos:

  1. Definir o nome da moldura de destino usando o atributo name na tag <frame> ou <iframe>.
  2. No link, usar o atributo target com o nome da moldura.

Exemplo:

<a href="pagina.html" target="nome-da-moldura">Texto do link</a>

Para abrir uma seção específica numa moldura, usa o mesmo, adicionando o marcador na URL:

Exemplo:

<a href="pagina.html#marcador" target="nome-da-moldura">Link para seção</a>
