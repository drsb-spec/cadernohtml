Uso de Imagens

Tag <img />

A tag <img /> é usada para inserir imagens numa página HTML.

Explicação:

  • É uma tag vazia (não tem fechamento).
  • Serve para exibir imagens no navegador.

Atributos:

  • src → indica o caminho da imagem.
  • height → define a altura (em pixels).
  • width → define a largura (em pixels).
  • alt → mostra um texto alternativo se a imagem não carregar.

Exemplo:

<img src="imagem.jpg" width="300" height="200" alt="Descrição da imagem" />
