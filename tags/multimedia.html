Elementos de Multimédia

Vídeo

O elemento <video> … </video> é usado para inserir vídeos numa página HTML.

Explicação:

  • Permite mostrar vídeos diretamente na página.
  • Usa o atributo controls para exibir os botões de reprodução.
  • O atributo src indica o arquivo de vídeo.

Exemplo:

<video src="video.mp4" width="640" height="360" controls></video>

Áudio

O elemento <audio> … </audio> é usado para inserir sons ou músicas numa página HTML.

Explicação:

  • Permite reproduzir áudio diretamente na página.
  • Usa o atributo controls para mostrar os controles de som.
  • O atributo src indica o arquivo de áudio.

Exemplo:

<audio src="musica.mp3" controls></audio>
