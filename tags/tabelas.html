Tabelas em HTML

Tags principais

Explicação:

  • <table> → define a tabela.
  • <caption> → define a legenda da tabela.
  • <tr> → cria uma linha.
  • <colgroup> → agrupa colunas para formatação.
  • <col> → define propriedades de colunas dentro de um colgroup.
  • <thead> → grupo de cabeçalhos.
  • <tbody> → grupo com o conteúdo da tabela.
  • <tfoot> → grupo com o rodapé.
  • <th> → define uma célula de cabeçalho.
  • <td> → define uma célula de dados.

Atributos importantes

Explicação:

  • align → alinha o conteúdo horizontalmente (left, right, center).
  • valign → alinha verticalmente (top, middle, bottom).
  • bgcolor → define a cor de fundo.
  • border → define a espessura da borda (em pixels).
  • cellpadding → espaço entre o texto e a borda da célula.
  • cellspacing → espaço entre as células da tabela.
  • width → define a largura (em px ou %).
  • colspan → une células horizontalmente.
  • rowspan → une células verticalmente.
